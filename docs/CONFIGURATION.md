# Configuration

## Server (`~/data/frp/frps.yaml`)

Generated by `ft server init`:

```yaml
bindPort: 7000
auth:
  token: frp_xxxx
webServer:
  addr: 0.0.0.0
  port: 7500
  user: admin
  password: admin
log:
  to: /home/user/data/frp/frps.log
  level: info
```

After editing: `ft server reload` (restarts server).

## Client (`~/data/frp/frpc.yaml`)

Generated by `ft client init`:

```yaml
serverAddr: YOUR_SERVER_IP
serverPort: 7000
auth:
  token: frp_xxxx
log:
  to: /home/user/data/frp/frpc.log
  level: info
webServer:
  addr: 127.0.0.1
  port: 7400
proxies:
  - name: ssh_6022
    type: tcp
    localIP: 127.0.0.1
    localPort: 22
    remotePort: 6022
```

### Adding proxies

Edit the `proxies` list directly:

```yaml
proxies:
  - name: ssh_6022
    type: tcp
    localIP: 127.0.0.1
    localPort: 22
    remotePort: 6022
  - name: web_8080
    type: tcp
    localIP: 127.0.0.1
    localPort: 8080
    remotePort: 8080
  - name: rdp_3389
    type: tcp
    localIP: 127.0.0.1
    localPort: 3389
    remotePort: 3389
```

After editing: `ft client reload` (hot-reload, no disconnect).

## SSH Config

Add to `~/.ssh/config` for easy access:

```
Host myserver
    HostName YOUR_SERVER_IP
    User your_user
    Port 6022
```

Then: `ssh myserver`

## Paths

| File | Path |
|------|------|
| Server config | `~/data/frp/frps.yaml` |
| Client config | `~/data/frp/frpc.yaml` |
| Server log | `~/data/frp/frps.log` |
| Client log | `~/data/frp/frpc.log` |
| Binaries | `bin/{os}_{arch}/` or `~/.frp-tunnel/bin/` |
